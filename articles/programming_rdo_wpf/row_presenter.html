RowPresenter

RDO.WPF automatically maps a collection of DataRow objects into a collection of RowPresenter objects. RowPresenter contains row level presentation logic which can be consumed by view elements such as RowView, or can be used as data binding source by providing both data values and view states. RowPresenter does not aware any view elements except holding a reference to its associated RowView.

Data Value Access

You can access data values via the following APIs of RowPresenter class:

API Description
GetValue Gets the data value from specified column.
EditValue Enters the editing mode and change value for specified column.

These APIs can be used to provide data values as data binding source.

Row Level View States

Each RowPresenter object contains the following row level view states:

  • IsSelected: Indicates whether this row is selected.
  • IsCurrent: Indicates whether this row is current row. Only current row can be in editing mode.
  • IsEditing: Indicates whether this row is in editing mode. Editing of the row can be saved or cancelled. If this property is true, the IsCurrent property must also be true.
  • IsVirtual: Indicates whether this is a virtual row for inserting. Normally this row is displayed as an empty row at the end of the data grid.
  • IsInserting: Indicates whether this row is in inserting mode. If this property is true, the property of IsVirtual and IsEditing are also be true.
  • IsExpanded: Indicates whether this row is expanded with its Children.
