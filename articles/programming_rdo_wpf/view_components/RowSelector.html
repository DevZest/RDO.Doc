RowSelector

RowSelector represents the control that can perform row selection operation, as shown in FileExplorer sample:

Features

  • Command handlers for row selection operations.

Usage

Put the RowSelector inside a RowView, it will be wired up and work automatically.

You can add RowSelector via composite row binding, as demonstrated in FileExplorer sample:

...
<StackPanel Orientation="Horizontal">
    <dz:RowExpander Name="_rowExpander" VerticalAlignment="Center" />
    <dz:RowSelector>
        <StackPanel Orientation="Horizontal">
            <Image Name="_icon" Width="20" Height="20" Stretch="Fill" />
            <TextBlock Name="_textBlock" Margin="5,0" />
        </StackPanel>
    </dz:RowSelector>
</StackPanel>
...

You can also use RowSelector in control template of RowView, for example, the RowView.Styles.Selectable style.

Implemented Commands

Command Input Implementation
MoveUp Up Key Moves current row up.
MoveDown Down Key Moves current row down.
MoveLeft Left Key Moves current row left.
MoveRight Right Key Moves current row right.
MoveToPageUp PageUp Key Moves current row to one page up.
MoveToPageDown PageDown Key Moves current row to one page down.
MoveToHome Home Key Moves current row to the first row.
MoveToEnd End Key Moves current row to the last row.
SelectExtendedUp SHIFT-Up Key Selects multiple consecutive rows up.
SelectExtendedDown SHIFT-Down Key Selects multiple consecutive rows down.
SelectExtendedLeft SHIFT-Left Key Selects multiple consecutive rows left.
SelectExtendedRight SHIFT-Right Key Selects multiple consecutive rows right.
SelectExtendedPageUp SHIFT-PageUp Key Selects multiple consecutive rows to one page up.
SelectExtendedPageDown SHIFT-PageDown Key Selects multiple consecutive rows to one page down.
SelectExtendedHome SHIFT-Home Key Selects multiple consecutive rows to the first row.
SelectExtendedEnd SHIFT-End Key Selects multiple consecutive rows to the last row.
ToggleSelection SPACE Key Toggles the selection mode of current row.

Customizable Services

  • RowSelector.ICommandService: Your data presenter can implement this service to change the commands implementation of this class.
