Syntax
[AttributeUsage(AttributeTargets.Property, AllowMultiple = false)]
public sealed class InputAttribute : Attribute
Remarks

Apply this attribute on public settable string property, for sensitive data such as password. RunDbInit(String[]) will inject value for the property, by first looking from environment variable, then read from console if environment variable does not exist.
