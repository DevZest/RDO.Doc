Class EmptyDbAttribute

Indicates DbSessionProvider<T> implementation will return an empty database.

public sealed class EmptyDbAttribute : Attribute
Remarks

Database generation requires an empty database. DbInit tools will seek this attribute to determine if the DbSessionProvider<T> will provide an empty database to avoid table already exists error.
